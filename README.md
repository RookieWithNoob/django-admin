
# 手动步骤

## 第一步
下载源码到本地
```shell
git clone https://github.com/newpanjing/simpleui_demo
```

## 第二步
安装依赖包

```shell
pip install -r requirements.txt
```

## 第三步
```shell
python manage.py runserver
```

## 第四步
打开浏览器，在地址栏输入以下网址
> http://localhost:8000/admin

## 第五步
在用户名和密码的框框输入
+ 用户名：python
+ 密码：123456
